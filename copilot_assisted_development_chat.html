User Question
cuando ingreso datos correctos y le doy enviar me sale error de por favor ingresa un nombre valido
🤖
GitHub Copilot
El problema ocurre porque tienes dos manejadores para el evento `submit` en el formulario. El primero siempre muestra "¡Gracias por tu mensaje!" y resetea el formulario, y el segundo hace la validación. Ambos se ejecutan, lo que causa un comportamiento inesperado. Solución: Deja solo un manejador de submit que haga la validación y muestre el mensaje de éxito si todo es correcto. Aquí tienes el bloque corregido: ````html
🟠 portfolio.html +0 -0
// ...existing code... <script> // ...galería... // Manejo y validación del formulario de contacto (solo frontend) document.getElementById('contactForm').addEventListener('submit', function(e) { e.preventDefault(); const nombre = this.nombre.value.trim(); const email = this.email.value.trim(); const mensaje = this.mensaje.value.trim(); const formMsg = document.getElementById('formMsg'); formMsg.style.color = 'red'; // Validación básica if (nombre.length < 2) { formMsg.textContent = 'Por favor, ingresa un nombre válido.'; return; } const emailRegex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/; if (!emailRegex.test(email)) { formMsg.textContent = 'Por favor, ingresa un correo electrónico válido.'; return; } if (mensaje.length < 5) { formMsg.textContent = 'El mensaje debe tener al menos 5 caracteres.'; return; } formMsg.style.color = 'green'; formMsg.textContent = '¡Gracias por tu mensaje!'; this.reset(); }); // Limpiar mensaje de error al escribir en cualquier campo const form = document.getElementById('contactForm'); ['nombre', 'email', 'mensaje'].forEach(field => { form[field].addEventListener('input', () => { document.getElementById('formMsg').textContent = ''; }); }); </script> // ...existing code... ```` Elimina el primer manejador de submit y usa solo este. Así la validación funcionará correctamente.
